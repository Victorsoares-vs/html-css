Inserindo negrito e itálico

aqui temos a diferença de forma e significado.

Comandos de forma no html foram migrados pra o css, através da tag "style", porém aqui você usa classes de significados, ao invés disso.

Negrito e Itálico

Para deixar o texto em negrito, você usa a partícula "strong" (tag semântica), como no exemplo a seguir:
palavra de Deus.

Para Itálico, a partícula já será "Em" (tag semântica), como no exemplo:
Palavra de Deus.

Obs. tags que carregam significados para um elemento, são chamadas semânticas. as que apenas modificam forma, são chamadas de não semânticas, e já não são mais utilizadas.
